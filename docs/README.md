# 숫자 야구

### 구현할 기능 목록

1. 입력
    2. 서로 다른 3자리의 수
    3. 게임이 끝난 경우 재시작/종료를 구분하는 1과 2 중 하나의 수

2. 과정
    4. 컴퓨터가 1-9까지 임의의 수 3개를 선택
    4. 사용자가 입력한 숫자에 대한 결과 출력
        5. 같은 수가 같은 자리에 있으면 스트라이크
        6. 다른 자리에 있으면 볼
        7. 같은 수가 전혀 없으면 낫싱
    8. 잘못된 값을 입력할 경우 예외처리
    9. 사용자가 정답을 맞히면 게임 종료
    10. 게임 종료 후 재시작 또는 종료
